# 图书馆管理系统8

#### Description
基于jsp的图书馆管理系统8拥有两种角色，分别为管理员和学生，具体功能如下：

管理员：图书管理、用户管理、违规处理、权限管理、个人信息修改

学生：借阅图书、归还图书、借阅历史、处罚记录、个人信息修改
